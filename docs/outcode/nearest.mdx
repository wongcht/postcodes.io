sidebar_label: "Nearest"
title: "Find Nearest Outcode"
---

Returns a list of outcodes nearest to the specified outcode, ordered by proximity.

Each result has the same shape as a standard outcode lookup. If the input outcode is not found, the API returns a 404 response code.

## Endpoint

```javascript
GET https://api.postcodes.io/outcodes/:outcode/nearest
```

## Example Request

```
https://api.postcodes.io/outcodes/NW1/nearest
```

## Optional Query Parameters

| Parameter | Description                                                                   |
| --------- | ----------------------------------------------------------------------------- |
| `limit`   | Limits number of outcodes to return. Defaults to 10. Maximum allowed is 100.  |
| `radius`  | Sets search radius in metres. Defaults to 5,000m. Maximum allowed is 25,000m. |
